The new plugin for Tapuz Delivery orders from WooCommerce.

== Description ==

WooCommerce plug-in enables Tapuz delivery clients to order delivery pickups straight from the order details page within WooCommerce, without the need to log-in and submit each delivery manually in Tapuz’s web based ordering system. The plug-in saves time and potential human errors.
